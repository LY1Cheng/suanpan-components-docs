## 组件原理
使用该组件实现将美国边境口岸出入境数据集下发给后续组件使用


- 输出美国边境口岸出入境数据集，该数据集包含了美国-加拿大，美国-墨西哥边境口岸的车辆/人员出入境数据
- 数据描述
     - Port Name：口岸名称
     - State：所属州
     - Port Code：口岸编号
     - Border：所属边境
     - Date：日期
     - Measure：通过形式
     - Value：数量
     - Location：地点，经度以及纬度
