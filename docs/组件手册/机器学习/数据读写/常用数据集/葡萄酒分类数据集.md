## 组件原理
使用该组件实现将葡萄酒分类数据集下发给后续组件使用


- 输出葡萄酒分类数据集，描述178种葡萄酒的特征。
- 数据描述
    - fixed_acidity：固定酸度
    - volatile_acidity：挥发性酸度
    - citric_acid：柠檬酸
    - residual_sugar：残糖
    - chlorides：氯化物
    - free_sulfur_dioxide：游离二氧化硫
    - total_sulfur_dioxide：总二氧化硫
    - density：密度
    - pH：PH值
    - sulphates：硫酸盐
    - alcohol：酒精
