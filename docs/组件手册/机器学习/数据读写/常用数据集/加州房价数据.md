## 组件原理
使用该组件实现将加州房价数据下发给后续组件使用


- 输出加州房价数据，共有20640个观察样本，每条数据包含房屋以及房屋周围的详细信息。
- 数据描述
    - longitude：经度	
    - latitude：维度	
    - housing_median_age：房龄中位数
    - total_rooms：总的房间数	
    - total_bedrooms：卧室数目
    - population：人口
    - households：家庭数
    - median_income：收入中位数
    - median_house_value：房屋价格
    - ocean_proximity：和海的距离
